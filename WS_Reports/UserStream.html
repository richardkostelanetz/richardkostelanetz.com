Help Card: Authenticated Users Click Stream
This extremely powerful report allows you to drill down through every page viewed within every session undertaken by your authenticated site users. It can be particularly useful in assessing the behavioral trends of these key visitors, and can help in the planning of new site content. This report can also assist in the enforcement of acceptable use policies in an Intranet environment (allowing administrators to hone in on individual authenticated employees of an organization).

Vertical axis: Authenticated.
The name of the authenticated user being analyzed.

Horizontal axis: Hits (default sorting).
Hits to the site by the corresponding authenticated user. Changing the sorting options in Settings > Statistics will alter the horizontal axis to the new sort method. This report can be sorted/graphed by hits, bytes, sessions, pages and errors.

Red lines (if present):
HTTP errors (Page not found, server error, etc)

Authenticated User:
Name of the authenticated user being analyzed. Click the hyperlink to view the authenticated user's session click streams. The resultant report will open in a new browser window.

Hits (%):
Number of hits to the site by the corresponding authenticated user. (Percentage as a proportion of hits to the site by all authenticated users.)

Sessions:
Number of sessions undertaken by the corresponding authenticated user.

Mean time:
The average time spent on the website during each session undertaken by the corresponding authenticated user. It is represented in hours, minutes and seconds (HH:MM:SS).

Pages:
Number of pages viewed by the corresponding site authenticated user.

Errors:
Errors generated as a result of hits to the site by the corresponding authenticated user.
